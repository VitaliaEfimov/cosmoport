# cosmoport

JavaRush online internship test task.

## Getting Started

This is my implementation of JavaRush summer online internship test task.

### Technologies used:

* Spring MVC
* Spring Data JPA

### Required software

* TomCat9 server
* MySQL server

### Installing

```
mvn -DiskipTests=true clean install
```

## Running the tests

```
mvn test
```
